# How to run

1. Compile with ucanaccess files

```bash
javac -cp ".;lib/*" RegistrationForm.java
```

2. Run with ucanaccess files

```bash
java -cp ".;lib/*" RegistrationForm
```

3. To view the entries.

```bash
javac -cp ".;lib/*" accessDBviewer.java
```

then, run.

```bash
java -cp ".;lib/*" accessDBviewer
```
